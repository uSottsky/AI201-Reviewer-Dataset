# History of AI

### Early enthusiasm and great expecations (1952-1969)
* 1952: Samuel's checker-playing program uses reinforcement learning
* 1957: Rosenblatt's Perceptron Convergence Theorem
* 1958: McCarthy developed Lisp programming language used for the next 30 years
* 1959: Newell and Simon's General Problem Solver (GPS) - solved puzzles like a human being
* 1960: Widrow and Hoff - Adaptive Linear Element (Adaline)
* 1963: Microworlds - Blocks World - vision (Huffman, 1971), constraint propagation (Waltz, 1975), NLP (Winograd, 1972), planner (Fahlman, 1974)
* 1963: Winograd and Cowan - large number of neural elements can represent a concept
* 1965: Robinson's Resolution Principle for first order logic
