# Risks and Benefits of AI

Lethal autonomous weapons
* small group can deploy an arbitrarily large number of weapons against human targets defined by some recognition criterion

Surveillance and Persuasion
* monitor activities of individuals in a massive scale
* using machine learning, tailoring information flows through social media can modify political behavior (voting) (2016 onwards)

Biased Decision Making
* decisions biased by race, gender or other protected categories; data reflect bias

Impact on Employment
* jobs taken over by machines due to automation

Safety-Critical Applications
* accidents by autonomous vehicles; health care

Cybersecurity
* use of machine learning to lure users into using malware
* machine learning can create highly effective tools for personalized blackmail and phishing attacks
