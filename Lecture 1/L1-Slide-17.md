# AI Problem Domains

### Chess
excellent test bed for developing search algorithms
* > $10^{123}$ possible moves (brute-force exhaustive search will not work)
* mid-50's: Newell, Shaw, and Simon developed the first operational chess program
* 60's: Greenblatt Chess Program won a game in a regular chess tournament beating a Class C Player. - (US Chess Federation Class D - 1,200 pts to GM - 2,600 pts)
* 70's and 80's: Northwestern, Belle and HiTech systems achieved Senior Master's rating
* Deep Thought (Carnegie Mellon University) - first system to achieve Grand Master status
* 1996: Deep Blue lost to Kasparov (4-2)
* 1997: Deep Blue won over Kasparov

chess research has led to a number of search algorithms such as $\alpha-\beta$ search, $B^*$ search, singular-extension search
* many of these search concepts have found their way into everyday applications
