# Learning Agent

- any type of agent (model-based, goal-based, utility-based, etc.) may incorporate learning

![[Pasted image 20260314021202.png]]
### Components
- **performance element** – action selection
- **learning element** – making improvements to the knowledge components
- **critic** – measures how agent is doing and determines how performance element should be modified
- **problem generator** – suggests exploratory actions that lead to new and more informative experiences
