# Agent and Environment

**Agent**: a system that perceives its environment through its sensors and acts on that environment through effectors.

![[Pasted image 20260314003957.png]]
### **Examples of Agents**
1. **Human agent: sensors = eyes, ears, skin, nose, tongue; effectors = hands, legs, mouth, etc.**
2. **Robot: sensors = camera(s), laser range finder, microphone; effectors = wheels, robot arm, etc.**
3. **Softbot: input = bit strings; output = bit strings; environment = Internet (e.g. Google Search Engine)**
