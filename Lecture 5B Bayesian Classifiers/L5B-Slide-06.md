# Pattern Recognition – Feature Space Example

![[Pasted image 20260315184542.png]]

- features: mean $\mu$ and standard deviation $\sigma$ of intensity values
- line divides feature space into regions that correspond to class A or B
- unknown input has feature vector that falls on one side of the line determines the class of the pattern
- line is computed from the patterns (called training patterns) whose true classes are known
