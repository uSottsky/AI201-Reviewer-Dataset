# Features
![[Pasted image 20260315184256.png]]
> Reference: Slide 5 - Two cell/tissue images labeled (a) Class A: benign and (b) Class B: malignant.

**Identify features** (measurable quantities that make these two classes distinct): intensity ? size ? shape ? etc.

**Feature vector:** $x=[x_{1},x_{2},...,x_{l}]^{T}$
* $l$ features $x_{i}$, $i=1,...,l$
* features are random variables since measurements exhibit random variation (noise, pattern itself)
