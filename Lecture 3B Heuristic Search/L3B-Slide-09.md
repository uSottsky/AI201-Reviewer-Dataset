# Heuristic Search

Greedy Search:
• **not optimal**
• **not complete** since it could start off an infinite path and never return
• has worst-case space and time complexity of $O(b^{m})$, m = max. depth of the search tree
• actual space and time complexity are usually better, depending on the heuristic function
