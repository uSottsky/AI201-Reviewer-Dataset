# Heuristic Search

### Greedy Search
Always expands the node that is estimated to be closest to the goal.

**heuristic function**, *h* estimates the cost of the cheapest path to the goal state. 
- h(n) = estimated cost of the cheapest path from the state at node n to a goal state 
- *h*(goal) = 0

**Greedy Search** is a best-first search that uses h to select the next node for expansion.
