Heuristic Accuracy and Search Performance
If a collection of admissible heuristics, $h_{1},...,h_{m}$ is available for a problem, we can define
					$h(n)=max(h_{1}(n),...,h_{m}(n))$

If the heuristic function is so complex that computing its value for a node takes as long as expanding hundreds of nodes, a less accurate but simpler heuristic function may make the search cost lower.
