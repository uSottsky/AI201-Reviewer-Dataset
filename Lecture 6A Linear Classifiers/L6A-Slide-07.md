# Linear Classifiers

* **Previous lecture:** classifiers based on probability density or probability functions resulting classifiers - equivalent to a set of linear discriminant functions
* **Today:** focus on the design of linear classifiers regardless of the underlying distributions describing the training data.

**Major Advantages of Linear Classifiers**
* simplicity
* low computational complexity

**Assumption:** all feature vectors can be classified correctly using a linear classifier.
When linear classifier cannot classify all vectors correctly, design optimal linear classifier by adopting appropriate optimality criterion.
