# 5. Static vs Dynamic

A **dynamic** environment changes while the agent is deliberating.  
It is necessary to keep sensing while deliberating.

If the environment does not change with time but the agent’s performance does, it is **semi-dynamic**.

**Examples**  
- chess → static  
- chess with clock → semi-dynamic  
- taxi driving → dynamic
