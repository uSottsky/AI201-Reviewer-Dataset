# Properties of Environments

### 5. Static vs Dynamic
A dynamic environment changes while the agent is deliberating.

It is necessary to keep sensing while deliberating for a dynamic environment.

If the environment does not change with passage of time but the agent's performance does, the environment is semi-dynamic.
* chess - static
* chess with clock - semi-dynamic (you could lose due to time default)
* taxi driving - dynamic
