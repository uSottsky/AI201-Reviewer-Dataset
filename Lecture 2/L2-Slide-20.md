# Properties of Environments

## 1. Fully vs Partially Observable

**Fully observable** environment  
- agent’s sensors give access to the complete state of the environment at each point in time  
- agent need not maintain internal state

**Partially observable** environment  
- part of state is not measured by sensors  
- noisy, inaccurate sensors
