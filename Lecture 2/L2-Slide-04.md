# Agent and Environment

### The Generic Intelligent Agent
A rational agent is an agent that acts rationally acts so as to achieve one's goals, given one's beliefs.

Necessary to measure achievement of goals. Must evaluate:
* how much successful the agent is
* when it was able to achieve its goal

**Performance Measure** indicates how successful the agent was

**Agent: Vacuum cleaning agent**
**Performance Measure**:
* amount of dirt cleaned up in a day
* consider also electricity consumed and noise generated
* when to evaluate performance is also important
    * surprise Exam: those with good study habits will tend to get high grades
