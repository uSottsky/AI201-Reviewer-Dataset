# Properties of Environments

### 3. Deterministic vs Indeterministic
An environment is deterministic if its next state is completely determined by the current state and the actions selected by the agent.

If the environment is partially observable, it may appear to be nondeterministic.

It is better to think of an environment as deterministic or indeterministic from the point of view of the agent
* example: chess deterministic
* poker - nondeterministic (since it is partially observable)
