# Types of Agents

### Model-based Utility-based Agent

![[Pasted image 20260314033933.png]]

Not all goals are the same
* goals may be viewed as world states which make the agent happy (e.g. taxi fare is low)

Objective of a utility-based agent is to maximize the degree of "happiness"
Utility - a function that maps a state onto a real number
* there are many goals, some of which are conflicting (e.g. speed vs safety).
* utility function provides the trade-off

Choose the action sequence that gives maximum utility.
