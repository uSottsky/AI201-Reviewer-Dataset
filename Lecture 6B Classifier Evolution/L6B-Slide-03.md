# Confusion Matrix

**Confusion Matrix** - useful for quantifying classifier performance on different classes
* **rows:** ground truth
* **columns:** classification results
![[Pasted image 20260316022919.png]]
**Accuracy** - total of correctly classified cases divided by the total number of cases in the test set
$$Accuracy = \frac{TP+TN}{TP+FP+TN+FN} = \frac{23+116}{23+5+116+12} = 0.891$$
