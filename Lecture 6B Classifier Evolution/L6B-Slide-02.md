# Accuracy

**Accuracy** - proportion of correct classifications on a given test set

$$Accuracy = \frac{\text{number of correct classifications}}{\text{number of samples in test set}}$$

* **balanced data:** accuracy indicates how good a classifier is
* **highly unbalanced data:** accuracy is useless
* misleading when higher cost is associated with a particular type of error

In healthcare, diseased cases are usually much less than healthy cases (highly imbalanced) so that overall accuracy is a poor metric.
