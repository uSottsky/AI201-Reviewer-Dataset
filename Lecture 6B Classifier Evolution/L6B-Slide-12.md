$$Recall = TP / (TP+FN)$$
$$Recall_P = 10 / (10 + 57) = 0.054$$
$$Recall_Q = 545 / (545 + 40) = 0.932$$
$$Recall_R = 11 / (11 + 261) = 0.040$$
$$Recall_S = 3 / (3 + 10) = 0.231$$

$$Balanced \ Accuracy = (0.054 + 0.932 + 0.040 + 0.231) / 4 = 0.314$$
