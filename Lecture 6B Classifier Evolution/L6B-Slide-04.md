# Precision and Recall

**Precision (P)** - number of cases that actually belong to class X divided by the number of cases that the classifier classifies as belonging to class X
$$P = \frac{TP}{TP+FP}$$

**Recall (R)** - number of cases belonging to class X that were correctly classified as class X, divided by the total number of class X cases.
$$R = \frac{TP}{TP+FN}$$

*Precision and Recall are in a tug of war - improving Precision sacrifices Recall and vice versa*
