# Specificity

**Specificity** - number of healthy cases ("class negative") that are correctly classified as such, divided by the total number of healthy cases
* quantifies how well the classifier is able to detect healthy cases from the healthy population
* low specificity means healthy cases are classified as disease cases and many "false alarms" are generated
* True Negative Rate

$$Specificity = \frac{TN}{TN+FP} = \frac{116}{116+5} = 0.959$$
