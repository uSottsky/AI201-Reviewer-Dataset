# Balanced Accuracy

**Balanced Accuracy** - alternative to accuracy for unbalanced test set
* **2 class:** average of Sensitivity and Specificity
$$Balanced \ Accuracy = \frac{Sensitivity + Specificity}{2}$$

* **multi-class:** average of Recall on each class
$$Balanced \ Accuracy = \frac{(Recall_1 + Recall_2 + ... + Recall_N)}{N}$$
