# Sensitivity

**Sensitivity** - number of diseased cases ("class positive") that are correctly classified divided by the total number of diseased cases
* quantifies how well the classifier is able to detect diseased cases from the diseased population and thus "raise an alarm"
* low sensitivity means many diseased cases are missed
* True Positive Rate
* = Recall

$$Sensitivity = \frac{TP}{TP+FN} = \frac{23}{23+12} = 0.657$$
