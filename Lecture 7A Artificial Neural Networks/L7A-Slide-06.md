# Biological Neurons

### Definition of ANN
An Artificial Neural Network is a massively parallel distributed processor that has a natural propensity for storing experiential knowledge and making it available for use.

A neural network resembles the brain in two aspects:
1. knowledge is acquired by the network through a learning process
2. inter-neuron connection strengths, known as synaptic weights, are used to store knowledge

Learning implies the ability to generalize:
* learn from an input training set and generalize from it
* **Generalization** refers to the neural network's ability to produce correct outputs for inputs not encountered during training.

A **Learning Algorithm** modifies the synaptic weights of the network in an orderly way so as to attain a desired design objective.
