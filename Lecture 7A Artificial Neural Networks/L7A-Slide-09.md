# Biological Neurons

**Prediction and Forecasting:** given a set of historical data $y(t_1), y(t_2), y(t_3), ... y(t_n)$ in a time sequence $t_1, t_2, t_3, ... t_n$, predict the sample $y(t_{n+1})$ at some future time $t_{n+1}$.
* Stock Market Prediction
* Weather Prediction

**Optimization:** find a solution satisfying a set of constraints such that an objective function is maximized or minimized
* Applications in science and engineering (e.g. Traveling Salesman Problem)

**Content-Addressable Memory:** access memory directly though its content, not through its address; content should be recallable even by a partial or distorted input
* multimedia information databases

**Control Systems:** generate a control input such that the system follows a desired trajectory determined by the reference model
* Process control
