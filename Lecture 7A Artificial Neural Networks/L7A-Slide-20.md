# Biological Neurons

### Network Architectures

![[Pasted image 20260316030318.png]]
> Reference: Slide 20 - Diagram of a Single Layer Feedforward Network. Input layer of source nodes fully connected directly to an Output layer of neurons.

Network architecture refers to how the neurons comprising the network are connected to one another.
* single vs multilayer
  * only computational units are counted when counting number of layers
* feedforward vs feedback

**Single Layer Feedforward Networks**
* signals flow forward (no feedback)
* example: linear associative network
