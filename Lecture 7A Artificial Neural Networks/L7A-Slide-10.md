# Biological Neurons

### Von Neumann Computer vs Biological Neural System

| Feature | Von Neumann Computer | Biological Neural System |
| :--- | :--- | :--- |
| **Processor** | • Complex<br>• High Speed<br>• One or several | • Simple<br>• Low Speed<br>• A large number |
| **Memory** | • Separate from Processor<br>• Localized<br>• Non-content addressable | • Integrated into Processor<br>• Distributed<br>• Content Addressable |
| **Computing** | • Centralized<br>• Sequential<br>• Stored Programs | • Distributed<br>• Parallel<br>• Self-learning |
| **Reliability** | • Very vulnerable | • Robust |
| **Expertise** | • Numerical & Symbolic Manipulations | • Perceptual Problems |
| **Operating Environment** | • Well-defined, well-constrained | • Poorly-defined, unconstrained |
