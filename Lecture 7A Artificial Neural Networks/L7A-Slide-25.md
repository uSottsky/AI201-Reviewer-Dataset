# Biological Neurons

### Definition of Learning
Learning is the process by which the free parameters (i.e., the synaptic weight values) of a neural network are modified through a continuing process of stimulation by the environment in which the network is embedded.

### Learning Paradigms

**Supervised Learning**
* modification of synaptic weights of a neural network by applying a set of labeled training samples or task examples
* each training sample consists of a unique input signal and its corresponding desired response

**Reinforcement (or Graded) Learning**
* similar to supervised learning except that, instead of being given the correct output, the network is given a score or grade that tells it how well it has done over a sequence of examples

**Unsupervised (or Self-Organized) Learning**
* learning from examples without the help of a teacher or critic
