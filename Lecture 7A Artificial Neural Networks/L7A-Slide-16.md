# Biological Neurons

### The XOR Problem

![[Pasted image 20260316030141.png]]
> Reference: Slide 16 - 2D graph of the XOR gate showing points $(0,0)$ and $(1,1)$ as Class B, and $(0,1)$ and $(1,0)$ as Class A. The points are arranged diagonally, making them not linearly separable.

| $x_1$ | $x_2$ | XOR Class |
| :--- | :--- | :--- |
| 0 | 0 | B |
| 0 | 1 | A |
| 1 | 0 | A |
| 1 | 1 | B |

Minsky and Papert (*Perceptrons: An Introduction to Computational Geometry*) proved that a single perceptron can't solve the XOR problem.
This led to the first AI winter.
