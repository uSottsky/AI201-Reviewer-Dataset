# Biological Neurons

### Learning Rules

* **Error-Correction Learning** - based on optimum filtering
* **Hebbian Learning** - inspired by neurological considerations
* **Competitive Learning** - inspired by neurological considerations
* **Boltzmann Learning** - based on thermodynamics and information theory

* Supervised Learning often employs the Error-Correcting Learning Rule to learn from the examples provided by a teacher.
* Unsupervised or Self-Organized Learning often employs the Competitive Learning Rule to discover patterns or features in the input data.
