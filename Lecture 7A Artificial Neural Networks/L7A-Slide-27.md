# Biological Neurons

### The Supervised Learning Paradigm

![[Pasted image 20260316030638.png]]
> Reference: Slide 27 - Block diagram of Supervised Learning. An Environment sends a state vector to both a Teacher and a Learning System. The Teacher produces a Desired response, the Learning System produces an Actual response. The difference is the Error signal, which loops back to adjust the Learning system.

* teacher possesses knowledge of environment in the form of input-output pairs
* neural network initially does not possess knowledge of environment.
* upon termination of learning process, the neural network is able to emulate the teacher
* teacher's knowledge of the environment is transferred to the neural network
* after training, the network can deal with the environment completely by itself
