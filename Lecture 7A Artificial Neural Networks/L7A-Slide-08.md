# Biological Neurons

### Tasks where ANNs are being used

**Pattern Classification:** assign an input pattern represented by a feature vector to one of many pre-specified classes
* Character Recognition - character symbols
* Speech Recognition - phonemes
* Face Recognition - identity of faces
* EEG waveform classification - normal/abnormal EEG
* blood cell classification - WBC, RBC
* PCB inspection - presence/absence of PCB defects

**Clustering/Categorization:** explore similarity between patterns and place similar patterns in a cluster
* Data Mining
* Data Compression
* Exploratory Data Analysis

**Function Approximation:** given a set of input-output pairs from an unknown function, find an estimate of the unknown function
* Modeling Problems
