# Problem Solving Agent

### Blocks-World
A planning domain in AI.

![[Pasted image 20260314131307.png]]
> Reference: Slide 8 - Diagram of the "Blocks-World" with colored cubes and a robotic gripper.

* **world**: numbered blocks that can be stacked to form a tower of unlimited height
* **goal**: stack block on each other according to a certain order
* stacking is achieved using a robot arm that has fundamental operations
* use logic and search to achieve goal; requires planning
* Sussman's Anomaly - subgoal interaction
