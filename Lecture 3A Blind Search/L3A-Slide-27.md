# Blind Search

### Two General Classes of Search Strategies
1. **Blind Search (Uninformed Search)**
    * no information is given about the number of steps or path cost from current state to the goal
    * choice on which node to expand depends only on the node's position in the search tree
    * **examples**: Breadth-First Search, Depth-First Search, Uniform Cost Search, Depth-Limited Search, Iterative Deepening Search, Bidirectional Search
2. **Heuristic Search (Informed Search)**
    * uses problem-specific knowledge to find solutions more efficiently
    * **examples**: Greedy Search, $A^{*}$ Search, Iterative Deepening $A^{*}$, Simplified Memory Bounded $A^{*}$, Real Time $A^{*}$, Learning Real Time $A^{*}$, $B^{*}$
