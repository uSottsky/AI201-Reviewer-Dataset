# Search for Solutions 
### Search Strategy Evaluation Criteria


Search strategies are evaluated according to: 
1.  **Completeness**: is it guaranteed to find a solution when there is one ? 
2. **Time Complexity**: how long does it take to find a solution ? 
3. **Space Complexity**: how much memory does it need to perform the search ? 
4. **Admissibility (Optimality)**: does it find the highest quality solution when there are several solutions ?
