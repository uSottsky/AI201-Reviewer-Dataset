# Problem Types

### Other Important Terms
Given a particular state x:
* **successor function** $S(x)$ returns the set of states reachable from x by any single action
* **state space** - the set of all states reachable from the initial state by any sequence of actions
* **path (in state space)** - any sequence of actions leading from one state to another
* **Goal** - a set of goal states or an abstract description (e.g. in chess, the goal is a checkmate - opponent's king can be captured on the next move no matter what the opponent does)
