# Search for Solutions

### Data Structures for Search Trees 

Make a node with the following data structure components: 
1. state in state space to which the node corresponds 
2. **parent node** - node in the search tree that generated this node 
3. **operator** that was applied to generate the node 
4. *depth of the node**: number of nodes on the path from the root to this node 
5. **path cost** of path from the initial state to the node 

**EXPAND** function takes care of calculating each of the components of the nodes it generates.
