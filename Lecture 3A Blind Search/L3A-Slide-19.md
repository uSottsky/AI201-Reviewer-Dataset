# Search for Solutions

### Searching for Solutions
**Solution** - a path from the initial state to a state that satisfies the goal test

**Effectiveness of search can be measured according to the following**:
* **Completeness** (Does it find the solution when there is one ?)
* **Optimality** (Does it find the lowest possible path cost ?)
* **Time Complexity** - cost associated with time needed to arrive at the solution
* **Space Complexity** - cost associated with memory needed to arrive at the solution

Once we have defined a problem (i.e. determined the initial state, the operators described by a successor function, specified the goal test, and formulated the path cost function), the solution can be found by a search through search space.
