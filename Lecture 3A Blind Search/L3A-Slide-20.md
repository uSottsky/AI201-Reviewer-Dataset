# Search for Solutions

### Generating action sequences
* **expand a state** - generate a new set of states by applying all valid operations to that state

![[Pasted image 20260314131831.png]]
> Reference: Slide 20 - Search tree diagram showing: (a) Initial state Arad, (b) After expanding Arad, (c) After expanding Sibiu.

**Informal steps of the search process**:
1. choose one state
2. expanding that state
3. apply goal test to the expanded states to check if any of them is the goal state and exit if goal was found
4. go back to step 1
