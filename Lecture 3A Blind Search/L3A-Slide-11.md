# Problem Types

### Single-State Problem

![[Pasted image 20260314131436.png]]
> Reference: Slide 11 - Grid of vacuum world states.

In a single-state problem the agent:
* knows exactly which state it is in (world is accessible)
* knows exactly the effects of its actions

If the agent knows that it is in world state 5, the action sequence {move-right, suck-dirt} brings it to the goal state.
