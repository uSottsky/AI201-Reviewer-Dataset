# Is there a Best Algorithm for Machine Learning ?

![[Pasted image 20260315165235.png]]
> Reference: Slide 24 - Graph plotting "performance" against "type of problem". A "highly specialized algorithm" peaks sharply for one problem but performs poorly elsewhere. A "general-purpose algorithm" performs at a flat average across all problems.

**No Free Lunch Theorem (Wolpert and Macready, 1997)**
There is no "best" machine learning algorithm that works well for all problems. Each carries a set of assumptions that works well in one domain but may work poorly in another.

* need to know many different types of models to cover the wide variety of data that occurs in the real world
* for each model, there may be many algorithms we can use to train the model where we make speed-accuracy-complexity tradeoffs.

*"All models are wrong, but some models are useful" (George Box, 1976)*
