# Classification

**Goal of Classification:** Learn a mapping from inputs to outputs $y$ where $y \in \{1, \dots, C\}$ where $C$ is the number of classes.

* **Binary Classification:** $y \in \{0,1\}$
* **Multi-Class Classification:** $y \in \{0, 1, \dots, n\}, n \ge 2$
* **Multi-Label Classification:** class labels are not exclusive (e.g., male and tall)
  * best viewed as predicting multiple related binary class labels (multiple output model)
