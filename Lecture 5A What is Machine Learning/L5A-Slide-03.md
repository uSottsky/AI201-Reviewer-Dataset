# Definition of Learning

**Learning:** a machine learns with respect to a particular task $T$, performance metric $p$, and type of experience $E$, if the system reliably improves its performance $p$ at task $T$, following experience $E$.

Depending on how we specify $T$, $p$, and $E$, the learning tasks may be called by names such as:
* data mining
* computer vision
* natural language understanding
* information retrieval
* etc.
