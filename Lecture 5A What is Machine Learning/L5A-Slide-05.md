# Learning Modalities

Given a machine that experiences a series of sensory inputs $x_{1}, x_{2}, x_{3}, \dots$

* **Supervised Learning:** the machine is also given desired outputs $y_{1}, y_{2}, y_{3}, \dots$ and its objective is to produce the correct output given a new input.
* **Unsupervised Learning:** the machine aims to build a model of $x$ that can be used for reasoning, decision making, prediction, communication, etc.
* **Reinforcement Learning:** the machine produces actions $a_{1}, a_{2}, \dots$ that affect the state of the world and receives rewards (or punishments) $r_{1}, r_{2}, \dots$. It aims to learn to act in such way that the overall reward is maximized in the long term.
