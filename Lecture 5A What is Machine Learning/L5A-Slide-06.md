# Supervised Learning

**Classification:**
* desired outputs $y_{i}$ are discrete class labels
* objective: classify new (unseen) inputs correctly (generalization)

**Regression:**
* desired outputs $y_{i}$ are continuous values
* objective: predict output accurately for new (unseen) inputs (generalization)
