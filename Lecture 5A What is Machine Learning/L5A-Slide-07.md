# Unsupervised Learning

**Objective of Unsupervised Learning:** Build a model or find useful representations of the data
* clustering
* dimensionality reduction
* modeling the data density

**Where Unsupervised Learning is used:**
* structure discovery
* data compression
* outlier detection
* etc.
