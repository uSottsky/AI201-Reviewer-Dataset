# Types of Learning

* **Supervised Learning:** algorithms are trained on labelled examples where the desired output is known for a given input.
  * After training the supervised learning algorithm will exhibit generalization properties such that it can produce "correct" outputs for unseen inputs (inputs not found in the training set).
* **Unsupervised Learning:** algorithms process unlabelled examples in order to discover structures in the data (e.g., document clustering)
* **Reinforcement Learning:** algorithms that learn through interactions with an environment.
  * A reinforcement learning agent learns from the consequences of previous actions (experience) and choices, and aims to select actions that maximize the accumulated reward over time.
