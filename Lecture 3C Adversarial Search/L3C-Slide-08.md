# Mini-Max

### Mini-Max
2-player game (players are MAX and MIN)
* we want MAX to win
* MAX tries to maximize value of pay-off function
* MIN tries to minimize value of pay-off function
* MAX's turn at the start
* MAX will find a strategy that will lead to a winning state regardless of what MIN does

Consider the ideal case where there is no time limit.
* no time limit = can compute the entire search tree until we reach the terminal states
