# Monte Carlo Tree Search (MCTS)

For a given playout policy
* from what positions do we start the playouts?
* how many playouts per position?

**Pure Monte Carlo Search** - performs $N$ simulations starting from current state and track which of the moves from have the biggest win percentages
* **selection policy** that focuses on exploration of states that have few playouts and exploitation of states that have done well in the past playouts
