# Game Playing in AI

### Single-agent vs Multi-agent Games
* 2-person games in which the players alternate moves
* multi-agent games in which teams collaborate to beat their adversaries

Opponent introduces uncertainty since agent does not know what opponent is going to do.

**In general, uncertainty can arise from:**
* incomplete knowledge of the environment (e.g. robot navigating in an environment with an incomplete map)
* opponent who acts so as to divert the agent from reaching the goal
* computational constraints: one does not have enough time to calculate the exact consequences of each move
