# AI Problem Domains

### Go
more advanced test bed for developing reinforcement learning algorithms
* > $10^{361}$ possible moves
* 2007: Monte Carlo Tree Search
* 2010: MogoTW won against Catalin Taranu (5p)
* 2011: Zen won against Takemiya Masaki
* 2013: Crazy Stone beat Yoshio Ishida
* 2015: AlphaGo beat European Go champion Fan Hui
* 2017: AlphaZero beat Ke Jie (world rank 1)
* 2017: Leela Zero - free and open source Go program released
