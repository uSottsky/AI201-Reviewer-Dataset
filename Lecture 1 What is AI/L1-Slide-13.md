# History of AI

### Expert Systems (1969-1986)
* 1969: Buchanan et.al.'s DENDRAL - chemical analysis expert system hypothesized a substance's molecular structure; rivaled chemistry experts at this task
* 1975: Shortcliffe et.al.'s MYCIN - backward chaining expert system with knowledge base of 600 rules- identifies bacteria causing severe infections and recommends antibiotics with dosage adjusted for patient's body weight
* 1982: McDermott's R1 - first commercial expert system; rule-based production system
* 1982: Japanese Fifth Generation Project - create computers using logic programming and massively parallel computing
