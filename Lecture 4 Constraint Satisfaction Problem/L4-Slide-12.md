# What is CSP?

![[Pasted image 20260315013814.png]]
> Reference: Slide 12 -  A search tree for the 8-puzzle showing state transitions.  

**Blind/Heuristic/Adversarial Search**
* action sequences to reach goal
* objective: path to goal
* e.g., Breadth First Search, Depth First Search, $A^{*}$, $\alpha-\beta$ Pruning, etc.
* goal test - a function over states


![[Pasted image 20260315013840.png]]
> Reference: Slide 12 -  A CSP search tree for map coloring showing variables being assigned colors incrementally.


**CSP Search**
* variable assignments as goal: correct values assigned to the variables
* values must satisfy constraints
* goal test - set of constraints or rules that satisfy legal combination of values
