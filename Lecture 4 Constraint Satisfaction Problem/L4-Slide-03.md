# What is CSP?

### Blind/Heuristic/Adversarial Search
* **state** - data structure that supports goal test, evaluate, successor
* **goal test** - logical test $(yes/no)$

### Constraint Satisfaction Problem (CSP)
* **state** - variables $X_{i}$ whose values are from domain $D_{i}$
* **goal test** - set of constraints defining legal combinations of values for variables

**Blind/Heuristic/Adversarial Search solution:** action sequence from start to goal state

**CSP solution:** assignment of values to the variables that satisfy all the constraints on each and every variable
